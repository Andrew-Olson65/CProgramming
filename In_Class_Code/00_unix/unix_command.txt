mkdir <folder name> - make directory.

ls - list. Displays the content in the directory.

cd <folder name> - Change directory. Move to another directory.

code <file name> - Creates a file and opens it in a text editor.

pwd - print working directory

clear - removes everything from the terminal

clt+L - clears terminal

gcc - compiles the code in the terminal

./ - run the text/program

\ - escape character

\* - escape sequence

\n - New line

return 0 - exits the main function.

Steps to make new file - 1: mkdir <folder name>
2: cd <folder name>
3: code <file name.c>
